Why grid for page layout

Grid handles both rows and columns, making two-dimensional layouts straightforward. We name grid areas and assign regions to those areas for clarity and reordering.

Why flex for navigation

Navigation is one-dimensional. Flexbox excels at distributing list items in a row or column and switching direction at mobile for thumb-friendly targets.

Accessibility tweaks

  • Two fonts: Merriweather (headings) and Inter (body).
  • High-contrast Adobe-style palette.
  • Visible focus outlines and a skip link.
  • Touch targets ≥ 44px on mobile.

Quick tips

  1. Name grid areas clearly.
  2. Keep heading levels logical.
  3. Label landmarks via aria-labelledby.
  4. Test with keyboard and a screen reader.
